JavaGrande (IBM) Propsal: Java Array Package
sInitially define and implement Fortran 90-like arrays as 100% pure Java class library.
sOne class for each elemental data type and rank,
e. g., doubleArray2D, intArray1D, ComplexArray3D.
sCompiler optimizes/parallelizes the Array class method calls as intrinsic operations.
sExample:
doubleArray2D A, B, C;
for (i=0; i<m; i++)
  for (j=0; j<p; j++)
    for (k=0; k<n; k++)
      C.set(i,j,C.get(i,j)+A.get(i,k)* B.get(k,j));
