Simple Algorithms for Analyzing Receiver Classes
sClass Hierarchy Analysis [Dean et.al.95]
•Applicable to Statically-Typed
•For each method signature m, precompute for each class c in C, set of all mk’s that exist in the descendents of c: methods(c, m)
•Doesn’t look at program --- O(1)
•At s, possible methods are methods(sc, m)
A: mA,nA
B: mB
C: mC,pC
D: mD,nD
A a; B b; C c;
a.m; //methods={mA,mB,mC}
b.m; //methods={mB,mC} c.m; //methods={mC} c.p; //methods={pC}
a.p; //methods={pC}
c.n; //error
