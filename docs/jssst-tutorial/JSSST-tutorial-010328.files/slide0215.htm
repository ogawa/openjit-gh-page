sProblem: Given callsite s, Set of classes and their relations C, receiver class rC and method signature m, which method mk to invoke?
•Dispatch(s, C, rC , m) = mk
Class A {
  int m() {}
  int n() {}  }
Class B extends A {
  int m() {}  }
Class C extends B {
  int m() {}
  int p() {}  }
Class D extends A {
  int m() {}
  int n() {}  }
A: mA,nA
B: mB
C: mC,pC
D: mD,nD
