This is the program transformation and compile flow of OMPC++ .
A user writes a shared memory multithreaded program. The OpenC++ metaclass for program transformation is compiled with OpenC++ compiler, and the compiler containing program transformation is generated. A source program is transformed by the generated compiler into the program which added DSM function. The generated program is compiled by MPC++ and linking with MPC++ RTL and DSM library. Finally, the execution program with DSM support is generated
