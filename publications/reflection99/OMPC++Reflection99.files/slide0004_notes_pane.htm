SO, we propose providing the platform portability in parallel programming using reflection or Open Compiler.
We can support diverse programming models by language extension. We can easily support diverse environment with the only writing compiler metaclasses for each environment.
And good performance can be expected, if we properly design the system.
Since program transformation by Open Compiler can performed statically at the compile time or with runtime information, the optimization suited for runtime environments or applications can be applied.




What is the present most portable language? Probably, it is Java language. However, this Java language also has only the portability which JavaVM offers. That is, functional extensions or the optimization suitable for runtime environment cannot be performed.
