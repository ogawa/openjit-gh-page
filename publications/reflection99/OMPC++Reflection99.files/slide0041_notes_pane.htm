Next I explain briefly the processing of read and write accesses. The algorithm is basically the standard strrongly-coherent invalidation-based protocol with ownership transfer.
In read access, it checks whether to have the memory block accessed locally first. If not had, a copy is demanded from the owner of the block and the copy of the block is made locally. And it accesses to a memory.
In write access, it check whether to be the owner of the memory block to access first. If it is an owner invalidation will be invoked to the processor with a copy. If it is not an owner, ownership will be required of an owner. When if does not have the copy locally at this time, the memory block is also got. And it accesses to a memory.
