This is an example of the program transformation by OMPC++.
The left list is before the transformation, the right is after the transformation.
A shared object is declared like shared<double>.
Before and after write accessing to a share object, writestart and writeend are inserted.
Malloc statement is transformed into allocate method of a shared object.
Initialize, Finalize and mpc_main function are added by transformation executed by overridden FinalizeInstance method. Initialize function calls memory allocation, and Finalize function calls memory release method of each shared object. A mpc_main function is equivalent to the main function of an MPC++ program.
Mpc_main function calls initialization and finalization function, and user defined sdsm_init,main,done functions.
