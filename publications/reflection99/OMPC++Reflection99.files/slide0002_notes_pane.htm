Platform portability for parallel computers is required and must satisfy three points.
First, diverse environments must be supported. Unlike a normal computer there are many memory and communication models. And many PC/WS clusters using commodity hardware are developed.
Second, we must not only support platform portability  but also get high performance. Programs should be optimized for each platform on any platforms.
Third, programming models must not be restricted due to the portability.

Now, it is asked for platform portability in parallel execution environment. It is mentioned that unlike a normal computer there are many memory and communication models  with various parallel execution environment and PC/WS cluster using commodity hardware are developed.
Moreover it not only realizes portability, but it is the true portability to also attain high performance. That is, in order to realize platform portability, a performance must not fall victim. And programs must be optimized with the environment and executed on every platform.
Finally a programming model must not be restricted for the portability, and various programming models must be able to be supported.
