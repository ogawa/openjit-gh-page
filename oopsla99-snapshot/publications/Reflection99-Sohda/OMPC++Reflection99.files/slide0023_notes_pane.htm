In OMPC++ system, you can easily write shared-memory-style multithreaded programs, using the following 3 APIs.
sdsm_init() initializes shared data structure etc.
It is executed on one processor.
Sdsm_main() is the main routine run in parallel.
sdsm_done is finalization routine such as release of memory,  it is executed on one processor.
Compared with the conventional object based software DSM, it is easy to port programs.
