We show what transformation is executed in each method.
In TranslateInitializer, the size of a shared object and the information on a type are obtained.
The access check to a shared memory is inserted in TranslateAssign.
The routine of initialization / finalization is generated and inserted in FinalizaInstance. Specifically, the routine for calls of the allocation and release of a shared memory, initialization of memory management tables and usr wrote routine such as sdsm_main() is generated and inserted.
