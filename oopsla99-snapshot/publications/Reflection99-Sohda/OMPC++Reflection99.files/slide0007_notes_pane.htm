OpenC++ offers Compile-Time MOP. The compiler containing program transformation is generated by the OpenC++ metaprogram and processing this with the OpenC++ compiler. When the program of a base level is processed with this compiler, a program will be transformed and a suitable execution program will be generated.
MPC++ is the parallel programming language which extend C++ only by the template and inheritance. MPC++ provides the user-level thread, local/remote thread creation, local/remote method invocation, and remote memory access.
